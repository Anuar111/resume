# Resume

LaTeX source for my professional resume.

## Build

### Overleaf (recommended)

1. Download this repo as a ZIP, or push it to a new Overleaf project via the GitHub integration.
2. Set the compiler to **pdfLaTeX**.
3. Compile `resume.tex`.

### Local (pdfLaTeX)

```bash
pdflatex resume.tex
```

A second pass may be required if the cross-references change.

## Files

- `resume.tex` — single-source LaTeX resume.
- `.gitignore` — ignores LaTeX build artifacts.
